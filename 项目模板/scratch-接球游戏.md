# 🐱 Scratch接球游戏 — 完整教程

## 🎮 游戏介绍
这是一个简单的接球游戏：
- 玩家控制一个小猫
- 球从天上掉下来
- 接住球得1分，漏掉球减1分
- 尽可能得高分！

## 🛠️ 实现步骤

### 第1步：创建项目
1. 打开 scratch.mit.edu
2. 点击"创建"
3. 选择中文界面

### 第2步：删除默认角色
1. 右键点击默认的小猫角色
2. 选择"删除"

### 第3步：添加新角色
1. 点击右下角的"选择一个角色"
2. 选择一个可爱的角色（比如小猫）
3. 再添加一个球的角色

### 第4步：给小猫添加代码

点击小猫角色，拖入以下积木：

```
当绿旗被点击
重复执行
    面向[鼠标指针]方向
    移到x:(鼠标的x坐标) y:(y: -150)
```

**解释**：
- `面向[鼠标指针]方向`：让小猫面向鼠标
- `移到x:(鼠标的x坐标) y:(-150)`：让小猫跟随鼠标左右移动，但固定在屏幕下方

### 第5步：给球添加代码

点击球角色，拖入以下积木：

```
当绿旗被点击
将[分数]设为[0]
重复执行
    移到x:(随机数(-200)到(200)) y:(180)
    重复执行直到<y坐标 < -170>
        将y坐标增加(-5)
        如果<碰到[小猫]?>那么
            将[分数]增加(1)
            播放声音[pop]
            移到x:(随机数(-200)到(200)) y:(180)
        end
    end
    将[分数]增加(-1)
```

**解释**：
- `移到x:(随机数(-200)到(200)) y:(180)`：球随机出现在屏幕上方
- `将y坐标增加(-5)`：球往下移动
- `如果<碰到[小猫]?>`：检测是否接住球
- `将[分数]增加(1)`：接住球加1分
- `将[分数]增加(-1)`：漏掉球减1分

### 第6步：创建分数变量
1. 点击"变量"分类
2. 点击"创建一个变量"
3. 变量名：`分数`
4. 点击"确定"
5. 勾选"分数"让它显示在舞台上

### 第7步：添加背景
1. 点击舞台
2. 选择"背景"标签
3. 点击"选择一个背景"
4. 选择一个你喜欢的背景

### 第8步：添加音效
1. 点击球角色
2. 选择"声音"标签
3. 点击"选择一个声音"
4. 选择一个"pop"或"boing"音效

## 🎨 进阶改进

### 改进1：增加难度
让球越掉越快：
```
当绿旗被点击
将[速度]设为[5]
重复执行
    将[速度]增加(0.1)
    ...（其他代码）
    将y坐标增加((0) - (速度))
```

### 改进2：多种球
添加不同颜色的球，不同颜色得不同的分：
- 红球：3分
- 蓝球：2分
- 绿球：1分

### 改进3：生命系统
添加一个"生命"变量：
- 每漏掉一个球，生命减1
- 生命为0时游戏结束

### 改进4：最高分记录
添加一个"最高分"变量：
- 每次游戏结束时，如果分数大于最高分，更新最高分

## 🧠 学到了什么

通过这个项目，你学会了：
1. **事件**：当绿旗被点击
2. **循环**：重复执行
3. **条件判断**：如果...那么
4. **变量**：分数
5. **随机数**：随机位置
6. **碰撞检测**：碰到角色
7. **坐标系统**：x和y坐标
