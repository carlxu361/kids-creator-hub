## 📋 关卡详情

### 第1关 🏠 创客基地搭建
**目标**：认识电脑上所有工具，知道每个工具是干什么的
**时间**：30分钟
**任务**：
1. 打开终端（Terminal），认识命令行
2. 打开VS Code，认识代码编辑器
3. 打开Scratch网页版，看看别人做的游戏
4. 打开Claude Code，跟AI打个招呼
5. 打开OBS录屏软件，试试录一段屏幕

**完成标准**：能说出5个工具的名字和用途
**成就**：🏠 基地建造者

---

### 第2关 ⌨️ 打字小能手
**目标**：打字速度达到每分钟20个汉字
**时间**：每天15分钟，持续2周
**任务**：
1. 安装搜狗输入法（如果还没装的话）
2. 每天练习金山打字通10分钟
3. 用TypeRacer跟网友比赛打字
4. 给AI写一段自我介绍（至少100字）

**完成标准**：能流畅打出一段话，不用一直看键盘
**成就**：⌨️ 键盘侠（好的那种）

---

### 第3关 🐱 我的第一个Scratch游戏
**目标**：用Scratch做一个能玩的小游戏
**时间**：周末2-3小时
**任务**：
1. 打开 scratch.mit.edu，选择中文
2. 跟着官方教程做一个"会动的小猫"
3. 加入自己的创意（比如让小猫说话、换颜色）
4. 给游戏加一个"得分"系统
5. 分享给家人玩一玩

**完成标准**：有一个能玩的Scratch游戏，有角色、有交互
**成就**：🐱 游戏设计师

---

### 第4关 🎬 我的第一个视频
**目标**：录制并剪辑一个2分钟的视频
**时间**：周末2小时
**任务**：
1. 用OBS录制一段屏幕操作（比如展示你的Scratch游戏）
2. 用CapCut（剪映）剪辑视频
3. 加上字幕、背景音乐
4. 导出视频文件

**完成标准**：有一个完整的视频文件，有开头、有内容、有结尾
**成就**：🎬 视频创作者

---

### 第5关 ✍️ 我的第一个AI写作
**目标**：跟AI合作写一个短故事
**时间**：周末1-2小时
**任务**：
1. 打开Claude Code
2. 告诉AI你想写什么类型的故事
3. 跟AI一起讨论故事大纲
4. 你写一段，AI帮你改一段
5. 把故事保存下来

**完成标准**：有一个500字以上的完整故事
**成就**：✍️ 故事大王

---

### 第6关 🎯 Scratch进阶：狼人杀
**目标**：用Scratch做一个狼人杀游戏
**时间**：2-3个周末
**任务**：
1. 先画出狼人杀的游戏流程图
2. 设计角色（狼人、村民、预言家等）
3. 用Scratch实现基本逻辑
4. 加入AI对手（让电脑能"思考"）
5. 测试并修复bug

**完成标准**：能跟电脑玩一局狼人杀
**成就**：🎯 游戏策划师

---

### 第7关 🐍 Python入门：代码变魔术
**目标**：用Python画图和做简单游戏
**时间**：每周末2小时，持续3周
**任务**：
1. 在VS Code里写第一行Python代码
2. 用turtle画一个彩色图案
3. 做一个"猜数字"游戏
4. 做一个"文字冒险"游戏
5. 学会用AI帮你解释代码

**完成标准**：能独立写一个50行以内的Python程序
**成就**：🐍 代码魔法师

---

### 第8关 🎮 Godot游戏引擎：真正的游戏
**目标**：用Godot做一个2D游戏
**时间**：3-4个周末
**任务**：
1. 认识Godot界面
2. 跟着教程做一个"弹球"游戏
3. 加入自己的角色和关卡设计
4. 加入音效和背景音乐
5. 导出成可以发给朋友玩的游戏文件

**完成标准**：有一个用Godot做的、有多个关卡的游戏
**成就**：🎮 游戏工程师

---

### 第9关 🎥 视频进阶：剪辑大师
**目标**：制作一个5分钟的"教学视频"
**时间**：2个周末
**任务**：
1. 选一个你擅长的话题（比如"如何用Scratch做游戏"）
2. 写一个视频脚本
3. 录制屏幕操作+旁白
4. 用CapCut做高级剪辑（转场、特效、字幕）
5. 发布到B站或分享给朋友

**完成标准**：视频有清晰的结构，别人看了能学到东西
**成就**：🎥 知识UP主

---

### 第10关 🚀 独立项目：我的大作品
**目标**：独立完成一个完整项目（自选方向）
**时间**：4-6个周末
**任务**：
1. 写一份项目计划书（要做什么、怎么做、需要什么）
2. 设计原型（画草图或写文档）
3. 动手实现
4. 测试和改进
5. 做一个展示视频或PPT
6. 给家人/朋友做一次"产品发布会"

**完成标准**：有一个完整的、能展示的作品
**成就**：🚀 独立创客

## 💡 小贴士

1. **不要着急**：每个关卡可以慢慢来，重要的是学会，不是快
2. **多问AI**：遇到不懂的，大胆问Claude！它不会笑话你
3. **保存作品**：每个关卡的作品都保存好，以后回来看会很有成就感
4. **找人分享**：做完作品给家人朋友看看，他们的鼓励很重要
5. **允许犯错**：编程就是不断出错、不断修改的过程，这很正常！
