# 第1关 🏠 创客基地搭建

## 🎯 任务目标
认识电脑上所有工具，知道每个工具是干什么的。就像探险家出发前要先认识自己的装备一样！

## ⏰ 预计时间
30分钟

---

## 📖 任务步骤

### 任务1.1：打开终端（Terminal）
**什么是终端？**
终端就像一个"魔法对话框"，你可以用文字命令控制电脑。虽然看起来有点吓人，但它其实很听话！

**怎么做：**
1. 按 `Command + 空格键`，输入"终端"或"Terminal"
2. 按回车打开它
3. 你会看到一个黑色的窗口，里面有光标在闪烁

**试试这个：**
在终端里输入以下命令，每输一个就按回车：
```bash
echo "你好，我是小创客！"    # 让电脑显示一句话
date                        # 显示今天的日期
whoami                      # 显示你的用户名
```

**AI帮你理解：**
> 如果你不懂某个命令是什么意思，打开Claude Code，问它：
> "echo命令是干什么的？用小学生能懂的话解释"

---

### 任务1.2：认识VS Code
**什么是VS Code？**
VS Code是一个写代码的"超级笔记本"。它能帮你检查错误、给代码上颜色、还能直接运行程序。

**怎么做：**
1. 在终端输入 `code` 或者在启动台找到VS Code
2. 看看左边的侧边栏（文件、搜索、插件等图标）
3. 点击"文件" → "新建文件"
4. 输入一些文字，保存为 `hello.txt`

**小知识：**
> VS Code是全世界最受欢迎的代码编辑器，数百万程序员都在用它！

---

### 任务1.3：探索Scratch
**什么是Scratch？**
Scratch是MIT（一个很有名的大学）发明的编程工具。它用积木块来写代码，就像搭乐高一样！

**怎么做：**
1. 打开浏览器，访问 scratch.mit.edu
2. 点击右上角的地球图标，选择"简体中文"
3. 点击"创建"开始新项目
4. 看看左边的积木块分类（运动、外观、声音等）
5. 拖几个积木块到中间，看看会发生什么

**探索任务：**
- 看看首页别人做的游戏，玩一玩
- 找到一个你觉得很酷的项目，想想它是怎么做的

---

### 任务1.4：跟AI打个招呼
**什么是Claude Code？**
Claude Code是一个AI助手，你可以跟它聊天、问它问题、让它帮你写代码。它是你的"编程搭档"！

**怎么做：**
1. 在终端输入 `claude` 启动Claude Code
2. 跟它打个招呼，比如输入：
   ```
   你好！我是一个11岁的小学生，今天刚学编程，请多多指教！
   ```
3. 看看它怎么回复你

**试试跟AI聊天：**
```
你能用简单的话告诉我，什么是编程吗？
```

**小贴士：**
> AI就像一个什么都知道的朋友，但有时候它也会犯错。所以不要完全相信它说的每句话，要学会自己验证！

---

### 任务1.5：试试录屏
**什么是OBS？**
OBS是一个免费的录屏软件。你可以用它录下你在电脑上的操作，做成视频教程或游戏视频。

**怎么做：**
1. 打开OBS（在启动台搜索"OBS"）
2. 点击"来源"下面的"+"号
3. 选择"屏幕捕获"
4. 点击"开始录制"
5. 做一些操作（比如打开Scratch拖几个积木）
6. 点击"停止录制"
7. 找到录制的视频文件，播放看看
