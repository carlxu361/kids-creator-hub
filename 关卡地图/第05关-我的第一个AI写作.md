# 第5关 ✍️ 我的第一个AI写作

## 🎯 任务目标
跟AI合作写一个500字以上的完整故事！

## ⏰ 预计时间
周末1-2小时

---

## 📖 什么是AI写作？

AI写作不是让AI替你写，而是**你和AI一起写**！
- 你提供创意和想法
- AI帮你组织语言、扩展细节
- 你来决定最终的内容
- 就像有一个写作搭档一样

---

## 🛠️ 准备工作
1. 打开Claude Code
2. 准备一个你喜欢的故事主题

---

## 📖 教程：跟AI一起写故事

### 第1步：确定故事类型（5分钟）

先想想你想写什么类型的故事：
- 🗡️ 冒险故事（勇者打败怪兽）
- 🔍 侦探故事（解开谜题）
- 🚀 科幻故事（未来世界、太空旅行）
- 🧙 魔法故事（魔法师、精灵）
- 🏫 校园故事（学校里发生的事）
- 🐾 动物故事（拟人化的动物）

**跟AI对话：**
```
我想写一个故事，但不知道写什么好。
你能给我5个有趣的故事主题吗？
我喜欢[冒险/科幻/魔法...]类型的故事。
```

### 第2步：设计故事大纲（15分钟）

一个好的故事需要：
1. **主角**：故事的主人公是谁？
2. **背景**：故事发生在哪里？什么时间？
3. **问题**：主角遇到了什么困难？
4. **发展**：主角怎么解决问题？
5. **结局**：最后怎么样了？

**跟AI对话：**
```
我想写一个关于[你的主题]的故事。
主角是一个[描述你的主角]。
故事发生在一个[描述背景]的地方。
请帮我设计一个故事大纲，要有：
- 开头（介绍主角和背景）
- 中间（遇到问题和解决过程）
- 结局（最后的结果）
用中文写，要适合小学生读。
```

### 第3步：写故事开头（15分钟）

有了大纲，现在开始写故事。你可以：
1. 自己先写一段
2. 让AI帮你扩展或修改
3. 选择你喜欢的版本

**跟AI对话：**
```
请帮我写这个故事的开头（100字左右）。
要求：
- 语言生动有趣
- 能吸引读者继续看下去
- 适合小学生阅读水平
```

**或者自己先写：**
```
这是我自己写的开头，请帮我看看有什么可以改进的地方：
[粘贴你写的内容]
```

### 第4步：继续写故事（30分钟）

一段一段地写，每写完一段就让AI帮你：
1. 检查有没有错别字
2. 语言是否通顺
3. 有没有更生动的表达方式

**跟AI对话：**
```
请帮我继续写故事的下一段。
上一段讲到：[总结上一段内容]
接下来要：[告诉AI接下来的情节]
请写100字左右。
```

**或者挑战自己：**
```
我来写下一段，写完你帮我看看：
[粘贴你写的内容]
```

### 第5步：写结尾（10分钟）

结尾要给读者一个满意的结果，可以：
- 圆满结局（问题解决，皆大欢喜）
- 开放式结局（留下悬念）
- 反转结局（意想不到的结果）

**跟AI对话：**
```
我的故事快写完了，现在需要一个结尾。
前面的情节是：[总结整个故事]
请给我3个不同的结尾方案，让我选择。
```

### 第6步：修改和润色（15分钟）

把整个故事放在一起，让AI帮你：
1. 检查错别字和语法错误
2. 看看前后是否连贯
3. 有没有可以写得更好的地方

**跟AI对话：**
```
这是我写完的故事，请帮我：
1. 检查错别字
2. 看看语言是否通顺
3. 给我一些修改建议
[粘贴你的完整故事]
```

## 📝 写作模板

### 模板1：冒险故事
```
标题：《[你的标题]》

在一个[描述地方]的地方，住着一个叫[主角名字]的[主角身份]。

有一天，[发生了什么事]。[主角名字]决定[主角要做什么]。

在路上，他/她遇到了[困难1]。但是他/她没有放弃，而是[解决方法1]。

接着，又遇到了[困难2]。这次他/她[解决方法2]。

终于，[主角名字]到达了[目的地]。在那里，他/她发现了[重要发现]。

最后，[主角名字][做了什么]，[结局如何]。

从此以后，[后续影响]。
```

### 模板2：校园故事
```
标题：《[你的标题]》

[主角名字]是[学校名]的[年级]学生。他/她最喜欢[爱好]。

那天，[发生了什么事]。这让[主角名字]感到[情绪]。

[主角名字]决定[要做什么]。他/她找到了[朋友名字]一起[做什么]。

在过程中，他们遇到了[困难]。但是通过[方法]，他们[解决了什么]。

最后，[结果如何]。[主角名字]学到了[道理]。
```

## 🧠 思考题

1. 为什么写故事前要先设计大纲？
2. 跟AI一起写作，你觉得最好的方式是什么？
3. 你自己写的部分和AI帮你写的部分，哪个你更满意？为什么？
4. 如果下次再写故事，你会怎么改进你的写作过程？
