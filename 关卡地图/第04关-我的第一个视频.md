## 📖 教程：做一个游戏展示视频

### 第1步：写脚本（15分钟）

在开始录制之前，先想想你要说什么。一个好的视频需要有结构：

```
【开头】（10秒）
"大家好，我是XX，今天我要展示我用Scratch做的游戏。"

【介绍游戏】（30秒）
"这是一个接球游戏。玩家控制小猫，接住从天上掉下来的球。"
"每接住一个球得1分，漏掉一个球减1分。"

【演示游戏】（60秒）
一边玩一边解说：
"你看，小猫跟着鼠标移动..."
"哇，球掉得好快！"
"现在得分是5分！"

【结尾】（20秒）
"这就是我做的第一个游戏！"
"如果你也想学编程，可以试试Scratch。"
"谢谢观看，下次见！"
```

### 第2步：录制屏幕（20分钟）

1. 打开OBS
2. 设置录制：
   - 来源：屏幕捕获
   - 音频：麦克风（如果你想录旁白）
3. 打开你的Scratch游戏
4. 点击"开始录制"
5. 按照脚本，一边玩一边解说
6. 录完后点击"停止录制"

**录制小技巧：**
- 说话要清楚，不要太快
- 如果说错了，没关系，后面可以剪掉
- 多录几遍也没关系，选最好的一遍

### 第3步：剪辑视频（45分钟）

打开CapCut，按以下步骤操作：

#### 导入素材
1. 点击"新建项目"
2. 点击"导入"，选择你录制的视频
3. 把视频拖到时间线上

#### 剪掉不需要的部分
1. 播放视频，找到不需要的片段
2. 在开始和结束位置点击"分割"（剪刀图标）
3. 选中不需要的片段，按Delete删除

#### 添加字幕
1. 点击"文字" → "新建字幕"
2. 输入你说的话
3. 调整字幕的位置和大小
4. 设置字幕出现的时间

#### 添加背景音乐
1. 点击"音频" → "音乐"
2. 选择一首免费的背景音乐
3. 把音乐音量调低（不要盖过你的声音）

#### 添加开头和结尾
1. 用CapCut的"文字"功能添加标题
2. 可以加一个"谢谢观看"的结尾

### 第4步：导出视频（5分钟）

1. 点击右上角的"导出"
2. 选择分辨率（推荐1080p）
3. 点击"导出"
4. 等待导出完成

## 🎬 视频模板

### 模板1：游戏展示视频
```
[标题画面] 我的第一个游戏
[介绍] 游戏名称、玩法
[演示] 实际玩的过程
[总结] 学到了什么
[结尾] 谢谢观看
```

### 模板2：教程视频
```
[标题画面] 如何用Scratch做接球游戏
[步骤1] 首先，打开Scratch...
[步骤2] 然后，添加角色...
[步骤3] 接着，写代码...
[演示] 看，游戏做好了！
[结尾] 你学会了吗？
```

### 模板3：vlog风格
```
[开场] 大家好，今天我要...
[过程] 记录你在做什么
[感受] 说说你的想法
[结尾] 今天的分享就到这里
```
