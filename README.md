## 🚀 快速开始

### 方法1：使用快速启动脚本
1. 双击 `快速启动.command` 文件
2. 会自动打开项目导航页面和VS Code

### 方法2：手动打开
1. 打开浏览器，打开 `index.html` 文件
2. 按照关卡顺序开始学习

## 💡 学习建议

1. **不要着急**：每个关卡可以慢慢来，重要的是学会
2. **多问AI**：遇到不懂的，大胆问Claude！
3. **保存作品**：每个关卡的作品都保存好
4. **找人分享**：做完作品给家人朋友看看
5. **允许犯错**：编程就是不断出错、不断修改的过程

## 🤖 AI使用

在终端输入以下命令启动AI助手：
```bash
claude
```

然后可以跟AI对话，比如：
- "我想学Python，该怎么开始？"
- "帮我解释这段代码是什么意思"
- "我想做一个游戏，有什么建议？"
