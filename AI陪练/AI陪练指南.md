# 🤖 AI陪练指南 — 如何正确使用AI

## 📖 AI是什么？

AI（人工智能）就像一个什么都知道的朋友，但它不是真正的人。它通过学习大量的文字资料来回答你的问题。

### AI能做什么？
- ✅ 回答问题（像百科全书）
- ✅ 帮你写代码
- ✅ 帮你写故事、改作文
- ✅ 解释复杂的概念
- ✅ 帮你调试程序错误

### AI不能做什么？
- ❌ 不能替你思考
- ❌ 不能保证100%正确
- ❌ 不能知道你心里在想什么
- ❌ 不能替你做决定

---

## 🎯 核心原则：AI是搭档，不是代写

### ❌ 错误的使用方式
```
"帮我写一个狼人杀游戏"
"帮我写一篇作文"
"帮我做作业"
```
这样你什么也学不到！

### ✅ 正确的使用方式
```
"我想做一个狼人杀游戏，应该怎么设计？"
"我写了一篇作文，请帮我看看有什么可以改进的"
"这道数学题我不太懂，能给我一些提示吗？"
```
这样你才能真正学到东西！

## 🗣️ 跟AI对话的技巧

### 技巧1：说清楚你的需求
**不好的问法：**
```
帮我写代码
```

**好的问法：**
```
我想用Python写一个猜数字游戏，
电脑随机生成一个1-100的数字，
玩家输入猜测的数字，
电脑告诉玩家猜大了还是猜小了，
直到猜对为止。
请帮我写这个代码，并解释每一行的意思。
```

### 技巧2：告诉AI你的水平
```
我是一个11岁的小学生，刚开始学编程。
请用简单的话解释，不要太复杂。
```

### 技巧3：分步骤问
不要一次问太多，一步一步来：
```
第1步：先告诉我怎么创建一个游戏角色
第2步：然后告诉我怎么让角色移动
第3步：接着告诉我怎么检测碰撞
```

### 技巧4：让AI解释代码
当你看到一段不懂的代码时：
```
请逐行解释这段代码的意思：
[粘贴代码]
用小学生能懂的话解释。
```

### 技巧5：让AI帮你找错误
当程序出错时：
```
我的程序出了这个错误：
[粘贴错误信息]

这是我的代码：
[粘贴代码]

请帮我找出错误在哪里，怎么修改。
```

## 📝 提示词模板

### 模板1：学习新知识
```
我想学习[主题]。
我是一个[年龄]岁的[学生/初学者]。
请用简单的话解释：
1. [主题]是什么？
2. 为什么要学它？
3. 怎么开始学？
4. 给我一个简单的例子。
```

### 模板2：写代码
```
我想用[编程语言]写一个[功能描述]。
要求：
1. [具体要求1]
2. [具体要求2]
3. [具体要求3]

请帮我写代码，并逐行解释每行代码的意思。
```

### 模板3：调试错误
```
我的程序出了问题。

错误信息：
[粘贴错误信息]

我的代码：
[粘贴代码]

请帮我：
1. 找出错误在哪里
2. 解释为什么出错
3. 告诉我怎么修改
```

### 模板4：改进写作
```
这是我写的[作文/故事/文章]：
[粘贴你的内容]

请帮我：
1. 检查错别字和语法错误
2. 看看语言是否通顺
3. 给我一些修改建议
4. 告诉我哪些地方写得好
```

### 模板5：设计项目
```
我想做一个[项目类型]。
我的目的是[你的目的]。
我的水平是[你的水平]。
我的时间是[你的时间]。

请帮我：
1. 设计一个简单的方案
2. 列出需要的步骤
3. 推荐使用的工具
4. 给我一些注意事项
```

## ⚠️ 安全规则

### 规则1：保护个人信息
**永远不要告诉AI：**
- 你的真实姓名
- 你的学校名称
- 你的家庭住址
- 你的电话号码
- 你的照片
- 你父母的信息

**可以说：**
- "我是一个11岁的小学生"
- "我在学编程"
- "我喜欢做游戏"

### 规则2：AI可能犯错
AI说的不一定是正确的！特别是：
- 具体的数字和日期
- 历史事件的细节
- 科学知识的准确性
- 代码可能有bug

**怎么验证：**
- 问AI："你确定吗？"
- 自己查资料确认
- 试试看AI说的方法是否真的可行

### 规则3：不要过度依赖
- 不要让AI替你做所有事
- 自己先尝试，实在不行再问AI
- 要理解AI给你的答案，不要只是复制粘贴

### 规则4：遇到不舒服的内容
如果AI说了让你不舒服的话：
1. 不要继续那个话题
2. 告诉爸爸妈妈
3. 可以重新开始一个新对话

## 🎯 AI学习小任务

### 任务1：自我介绍（5分钟）
跟AI做一个自我介绍，练习如何描述自己：
```
你好！我是一个11岁的小学生。
我喜欢[你的爱好]。
我最近在学[你在学什么]。
我想用AI帮助我[你的目标]。
请记住我的信息，以后帮我时要考虑我的水平。
```

### 任务2：解释代码（5分钟）
找一段你不懂的代码，让AI解释：
```
请解释这段代码的意思：
print("Hello, World!")

用小学生能懂的话解释，最好举个例子。
```

### 任务3：创意头脑风暴（10分钟）
让AI帮你出主意：
```
我想做一个游戏，但不知道做什么好。
请给我5个有趣的游戏创意，要求：
1. 适合用Scratch做
2. 玩起来有趣
3. 难度适中
每个创意简单描述一下玩法。
```

## 💡 最后的建议

1. **多问"为什么"**：不要只要答案，要理解原因
2. **自己先尝试**：遇到问题先自己想，实在不行再问AI
3. **验证AI的答案**：特别是代码，要自己运行试试
4. **保持好奇心**：AI知道很多有趣的东西，多探索
5. **享受过程**：学习应该是有趣的，不要太有压力
